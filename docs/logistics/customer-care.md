# 客戶關懷

**位置**：物流作業 → 客戶關懷 Tab

此 Tab 管理維修完成後的後續追蹤，確認客戶滿意並建立良好的售後關係。

## 什麼時候需要操作這個 Tab？

- 工單已結案（`completed`）且設定了關懷追蹤日期
- 維修後第 7 天，進行客戶滿意度確認

## 列表說明

顯示設定了關懷日期的已完成工單，依日期排序提醒管理員行動。

每列顯示：
- RMA 號碼
- 客戶名稱
- 產品型號
- 維修完成日
- 關懷追蹤日期（到期日）
- 關懷狀態（未處理 / 已處理）

### 今日到期提示

當日應處理的關懷項目會以藍色標示，方便快速識別。

## 設定關懷追蹤

在工單出貨時，系統提示設定關懷日期：

1. 出貨後在工單詳情頁找到「後續關懷」區塊
2. 設定關懷日期（建議：出貨後 7 天）
3. 選擇性填寫「關懷備註」（如特殊問題需追蹤）

## 關懷追蹤行動

當追蹤日期到期時：

### 標準關懷流程

1. 在「RMA 回覆 Tab」或直接 Email 聯繫客戶
2. 確認：
   - 產品收到後是否正常使用？
   - 維修問題是否完全解決？
   - 有無其他問題需要協助？
3. 在關懷追蹤 Tab 點擊「標記已處理」
4. 填寫追蹤結果備註

### 若客戶反映問題未解決

1. 先確認是否為同一故障問題
2. 若是同一問題：開新 RMA 工單，可在備註關聯原 RMA 號
3. 若是新問題：正常申請新 RMA，說明保固狀態

## 滿意度調查（選用）

部分工單可設定寄送滿意度調查問卷：

1. 在工單詳情頁「後續關懷」區塊啟用問卷
2. 設定問卷寄送時間
3. 系統自動在指定時間寄送調查 Email

> 目前問卷功能需主管開通才可使用，預設關閉。

## 注意事項

- 關懷追蹤是可選的，但建議對高價值客戶（如代理商、多次申請的客戶）務必執行
- 若客戶主動回饋好評，可截圖留存供行銷參考
- 關懷結果（滿意 / 不滿意）會影響 Dashboard 的客戶滿意度統計
