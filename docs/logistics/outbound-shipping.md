# 出貨處理

**位置**：物流作業 → 出貨處理 Tab

此 Tab 管理產品維修完成後，寄回給客戶的出貨作業。

![出貨處理 Tab](/rma-helper-hub/screenshots/logistics-outbound.jpg)

## 什麼時候需要操作這個 Tab？

- 維修完成，準備寄回給客戶
- 需要填寫快遞商與追蹤號碼

## 列表說明

顯示所有維修完成（`repairing` 完成）、待出貨的 RMA 工單。

每列顯示：
- RMA 號碼
- 客戶名稱 / 收件地址
- 產品型號
- 完修日期
- 出貨狀態

## 出貨流程

1. 包裝產品（確認所有配件齊全）
2. 選擇快遞商並取得追蹤號碼
3. 在此 Tab 找到對應 RMA，點擊「填寫出貨資訊」
4. 填寫：
   - **快遞商**（如黑貓宅急便、順豐速遞）
   - **追蹤號碼**
   - **寄出日期**（預設今天）
5. 儲存後工單自動更新為 `completed`
6. 系統傳送「維修完成通知」Email 給客戶（含追蹤號碼）

## 出貨注意事項

- 確認收件地址與客戶申請時一致（如有更改，需更新地址後再出貨）
- 包裝需使用適當的保護材料（潛水電腦錶屬於精密儀器）
- 保留寄件憑單（有追蹤號即可，無需另存圖片）

## 批次出貨（含供應商維修批次）

供應商維修批次完成後，會在「供應商維修 Tab」確認驗收，驗收後的工單會自動出現在此 Tab。

若同一批有多台需寄出：
1. 在出貨處理 Tab 選取多筆工單
2. 點擊「批次出貨」
3. 填寫快遞商（各台追蹤號可分別填寫）

## 地址確認

若客戶地址有誤或需變更：
1. 前往 RMA 工單詳情頁
2. 確認或修改收件地址
3. 回到出貨處理 Tab 繼續出貨

> 重要：請勿對地址有疑問的工單直接出貨，先確認地址再寄件。
