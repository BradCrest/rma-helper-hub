# 搜尋與篩選

## RMA 列表（/admin/rma-list）

### 搜尋欄位

頁面頂端搜尋框支援關鍵字搜尋，可匹配：
- **RMA 號碼**（如 `RMA-20250503-001`）
- **客戶姓名**
- **客戶 Email**
- **序號**（Serial Number）

> 搜尋為即時篩選（不需按 Enter），輸入後列表立即更新。

### 狀態篩選

點擊狀態 Badge 按鈕，可快速篩選特定狀態：

| 按鈕 | 篩選結果 |
|------|---------|
| 全部 | 顯示所有 RMA |
| 已登錄 | `registered` |
| 已寄出（客→公司）| `shipped` |
| 已收到 | `received` |
| 檢測中 | `inspecting` |
| 聯繫客戶中 | `contacting` |
| 已確認方案 | `quote_confirmed` |
| 已付款 | `paid` |
| 不維修 | `no_repair` |
| 寄回新品 | `shipped_back_new` |
| 寄回整新機 | `shipped_back_refurbished` |
| 寄回原機 | `shipped_back_original` |
| 後續追蹤 | `follow_up` |
| 已結案 | `closed` |

### 日期篩選

可篩選「建立日期」區間：
- 開始日期 / 結束日期
- 快捷：本週 / 本月 / 上個月

### 進階篩選

| 篩選條件 | 說明 |
|---------|------|
| 客戶類型 | 一般消費者 / 經銷商 / 代理商 |
| 保固狀態 | 保固內 / 過保 / 無法判定 |
| 有未讀回覆 | 僅顯示有客戶未讀 Email 的工單 |

## 工單列表欄位說明

| 欄位 | 說明 |
|------|------|
| RMA 號碼 | 點擊可開啟詳情 |
| 客戶 | 姓名（客戶類型標籤）|
| 產品 | 型號 / 序號 |
| 故障類型 | 申請時選擇的問題類別 |
| 保固 | 保固狀態 Badge |
| 狀態 | 目前工單狀態 |
| 建立日期 | 申請時間 |
| 最後更新 | 最近一次狀態變更 |

## 工單詳情頁

點擊任一工單號，進入詳情頁，包含以下區塊：

### 基本資訊
- 客戶資料（姓名、Email、電話、地址）
- 產品資訊（型號、序號、購買日期）
- 故障描述（含申請人類型前綴）
- 照片附件

### 保固資訊
- 批次判斷結果
- 生產日期（由序號解碼）
- 保固到期日
- 手動覆蓋選項

### 狀態操作
- 狀態下拉選單（含備註欄）
- 「更新狀態」按鈕

### 物流資訊
- Inbound：客戶寄件快遞商、單號
- Outbound：寄回快遞商、單號

### 歷史記錄
- 所有狀態變更的時間軸
- 每次變更的操作者與備註

### Email 往返
- 此 RMA 的所有 Email 對話（thread 模式）
- 可直接在此頁面回覆

### Shopify 訂單
- 依客戶 Email 查詢 Shopify 購買記錄
- 用於核對購買日期（保固判斷依據）

## 排序

列表預設依「最後更新時間」降序排列（最新的在最上面）。  
點擊欄位標題可切換排序（升序 / 降序）。

## 匯出

> 目前版本尚未提供 CSV 匯出功能。若需要匯出，請使用 Supabase Dashboard 直接查詢 `rma_requests` 資料表。
