# 知識庫管理 & 客戶回覆

**位置**：`/admin/email-knowledge`（側邊選單：客戶回覆及知識庫）

此頁面整合三個功能：**知識庫建立**（供 AI 學習）、**RMA 回覆**（Email 往返管理）、**客戶來信**（Gmail 收件匣）。

---

## 📖 知識庫建立 Tab

知識庫是 AI 輔助回覆功能的核心。管理員上傳的文件、FAQ、範本，都會被向量化（Embedding），供 AI 在起草回覆時語意搜尋參考。

### 上傳知識文件

上傳流程：
1. 選擇語言（預設：繁體中文）
2. 輸入標籤（選填，例如：產品說明書、保固政策）
3. 拖放或點擊選擇檔案（支援 `.md` / `.txt` / `.eml` / `.pdf`，單檔最大 10MB）
4. 點擊「開始上傳」

> 系統自動將文件切段（chunking）並排入背景索引佇列。

### Gmail 自動同步（即將推出）

第二阶段將支援 Gmail OAuth 自動抓取信件，目前尚未開放。

### 向量化進度

頁面顯示 Email 知識庫向量索引狀態：
- 完成進度：`19320 / 19321 (100%)`
- 顯示待處理、處理中、完成、失敗筆數
- 背景工作狀態（最近啟動 / 心跳 / 完成時間）

若有失敗筆數，點擊「立即喚醒背景索引」重新觸發。

### 最近上傳的索引狀態

顯示過去 1 小時內上傳的文件，每 5 秒自動更新。

索引狀態說明：
- **pending**：排隊中或處理中
- **completed**：已完成，可被 AI 搜尋
- **failed**：向量化失敗（通常因 API 錯誤）

### 知識庫 AI 對話

右側面板提供 AI 問答介面，直接查詢知識庫內容：

| 模式 | 說明 |
|------|------|
| 直接提問 | 輸入問題，AI 根據知識庫回答 |
| 主動學習 | AI 提出建議問題，協助補充知識庫 |

**使用情境**：
- 確認知識庫是否涵蓋某個主題
- 測試 AI 回覆品質
- 快速查詢特定政策規則

### 知識庫建置建議

剛上任時，建議優先建立：

1. **保固政策 FAQ**（5-10 條）
   - 如何判斷保固期？
   - 過保費用怎麼計算？

2. **標準回覆範本**（5-10 條）
   - 申請確認回覆
   - 請客戶提供購買憑證
   - 維修完成通知
   - 整新品報價

3. **常見故障問答**
   - 螢幕問題處理
   - 電池問題說明

## 💬 RMA 回覆 Tab

針對特定 RMA 工單的 Email 往返管理，支援 AI 輔助起草回覆。

### 介面說明

- **左側**：RMA 列表（搜尋 RMA 號碼 / 客戶 / 產品），可依狀態篩選
- **右側**：Email 對話 thread + AI 起草區

### 使用流程

1. 在左側搜尋找到目標 RMA 工單
2. 點擊工單進入右側對話面板
3. 查看完整的 Email thread 記錄
4. 點擊「AI 起草回覆」讓系統根據知識庫生成草稿
5. 修改草稿內容
6. 確認後點擊「傳送」

### AI 起草品質優化

若 AI 起草品質不佳：
1. 至「知識庫建立 Tab」補充相關知識文件
2. 確認知識文件的向量化狀態為 `completed`
3. AI 會自動從最新知識庫重新搜尋

### 常用附件插入

傳送回覆時，可一鍵插入「常用檔案庫」中的檔案（如保固政策 PDF、報價單範本），不需重新上傳。

## 📬 客戶來信 Tab

整合 Gmail，在系統內直接處理客戶來信，不需另開 Gmail。

### 連線狀態

頁面頂部顯示：
- 🟢 **已連線**：Gmail 授權有效，可正常讀取信件
- 🔴 **未連線**：需重新授權

### 介面說明

| 區域 | 說明 |
|------|------|
| 左側信件列表 | 全部 / 未讀 切換；支援 Gmail 搜尋語法（如 `from:xxx@gmail.com`）|
| 右側閱讀區 | 信件內容顯示 + 回覆區 |
| 同步按鈕 | 手動觸發同步最新信件 |

### 搜尋語法

支援標準 Gmail 搜尋語法：
- `from:客戶@email.com` — 篩選寄件人
- `subject:RMA` — 篩選主旨
- `is:unread` — 只顯示未讀

### 處理流程

1. 切換至「未讀」查看新信件
2. 點擊信件進入右側閱讀
3. 確認信件是否與某筆 RMA 相關
4. 若相關：在「RMA 回覆 Tab」找到對應工單回覆（保持 thread 完整）
5. 若不相關：在此直接回覆

> **建議**：盡量透過「RMA 回覆 Tab」回覆，確保信件記錄與工單綁定，方便日後查詢。
