# 首次登入

## 登入入口

前往：**[https://rma-helper-hub.lovable.app/admin](https://rma-helper-hub.lovable.app/admin)**

輸入管理員 Email 與密碼，點擊「登入」。

> **注意**：登入頁面是公開的，但登入後的 `/admin/*` 所有頁面都需要有效的 admin 角色，否則會被重定向到登入頁。

## 角色說明

系統有兩種管理員角色：

| 角色 | 說明 |
|------|------|
| `admin` | 一般管理員，可操作所有 RMA 功能 |
| `super_admin` | 超級管理員，額外可管理其他管理員帳號、查看登入記錄 |

## 申請新帳號流程

1. 新管理員自行前往登入頁，使用 Email 完成 Supabase Auth 註冊
2. 系統自動將申請記錄寫入 `pending_admin_registrations` 佇列
3. **`super_admin`** 在設定頁審核並核准
4. 核准後，新管理員重新登入即可正常使用

> 在核准之前，即使登入成功，也會因為沒有 `user_roles` 記錄而被拒於後台外。

## 登出

點擊畫面右上角的帳號圖示，選擇「登出」。  
每次登入/登出都會記錄於 `login_logs`（含 IP、地理位置、瀏覽器資訊）。

## 忘記密碼

目前需由 `super_admin` 在「設定 → 管理員管理」中執行「重設密碼」，系統會寄送重設連結至該 Email。

## 安全提醒

- 請勿在公用電腦保存登入狀態
- Session 一段時間未操作後會自動過期，需重新登入
- 若發現異常登入，請立即通知 `super_admin` 並更改密碼
